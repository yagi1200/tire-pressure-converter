# 空気圧 単位換算ツール

kPa（キロパスカル）／ bar（バール）／ psi（プサイ）／ kgf/cm²（キログラムフォース毎平方センチ）を相互に換算するシンプルなWebツール。

数値を入力して単位を選ぶと、他の3単位へ同時に換算します。オフロードバイク（kgf/cm²）とマウンテンバイク（bar / psi）の両方で使う想定。手持ちの空気入れ・エアゲージ（kgf/cm²・kPa 表記）からすぐ変換できます。

## 使い方

`index.html` をブラウザで開くだけ。ビルド・依存関係なしの単一HTMLファイルです。

## 換算基準（1単位 = ? kPa）

| 単位 | kPa換算 |
|------|---------|
| 1 bar | 100 kPa |
| 1 kgf/cm² | 98.0665 kPa |
| 1 psi | ≈ 6.89476 kPa |

※ 表示はゲージ圧（大気圧を0とした値）です。

## クイックプリセット

オフロード前後・MTBトレイル/舗装寄り・ロード目安・低圧グリップをワンタップ呼び出し。
